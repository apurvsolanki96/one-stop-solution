NOTAM Parser — Offline (Tested)

Standalone browser tool. Load your file and run diagnostics. No network, no backend changes.
This tool is offline-only. Keep it in a separate folder to avoid replacing your production index.html.
How this helps you: After loading your `THE AWY OUTPUT ONLY.TXT`, click Parse & Run Diagnostics on Loaded File. The tool will attempt to parse every example and report which items were parsed cleanly and which need improved rules. Copy any failing examples and paste them here — I'll update the parser.
Tip: Keep this folder separate from your deployment. If you want me to harden the parser to exact lines, paste failing lines here and I’ll update the parser file.
